Problem PC: покрытие точек отрезками

У вас есть n точек с целыми координатами на прямой. Их необходимо покрыть отрезками длины unitlen, используя минимальное количество отрезков (ваш запас отрезков в этой задаче неограничен, но все они одинаковые)

Отрезок покрывает точку в том числе, если он совпадает с ней в начальной или конечной координате, например отрезок (0, 2) покрывает точки 0, 1 и 2

Нужно вернуть количество использованных отрезков, получившееся после минимизации

int coverpoints(int *pts, int n, int unitlen) {
  // TODO: ваш код здесь
}

Вы не должны полагаться на то, что входной массив точек упорядочен


Пример

Для приведённого на рисунке множества точек и длины единичного отрезка вы должны вернуть ответ 2


Ещё пример:

unitlen = 1

Точки: 2 3 4 1 6 5

Ответ: 3 отрезка (1, 2), (3, 4), (5, 6)
