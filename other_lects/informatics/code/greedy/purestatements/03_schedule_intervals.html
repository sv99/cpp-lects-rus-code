Problem IC: расписание аудитории

Для конкретной аудитории есть ряд заявок с номером заявки (number) временем начала (start) и временем окончания (fin)

struct intvl_t { int number; int start; int fin; };

Назовём заявки конфликтующими если их временные интервалы пересекаются

Например (0, 2) и (1, 3) конфликтующие

Также (1, 2) и (2, 3) конфликтующие, то есть достаточно пересечения в одной точке

Вы должны разработать функцию, которое по заданному множеству заявок возвращает количество не конфликтующих

int schedulemax (struct intvl_t *reqs, int nreqs) {  
  // TODO: ваш код здесь
}

Вы должны быть готовы к тому, что пришедший на вход массив интервалов никак не отсортирован


Пример

Красным помечено выбранное оптимальное множество. Для такого множества вы должны вернуть число 6
